# 🛡️ CyberSentinel - AI-Powered OSINT & Cybersecurity Intelligence Platform

CyberSentinel is a professional Open Source Intelligence (OSINT) and Cybersecurity Investigation Platform designed for cyber researchers, security analysts, investigators, and students. The platform provides a unified interface to collect, analyze, and visualize intelligence from publicly available sources, helping users identify digital footprints, assess risks, and investigate domains and individuals.

> **Development Status:** Active Development
> Some investigation modules are currently under development and may use simulated or limited data while real-time API integrations are being enhanced.

---

## 🚀 Key Features

### 🌐 Domain Intelligence

Analyze domain names to uncover infrastructure, ownership, and security-related information.

#### Features

* WHOIS Lookup

  * Registrar Information
  * Domain Creation & Expiry Dates
  * Registrant Details

* DNS Enumeration

  * A Records
  * MX Records
  * TXT Records
  * NS Records
  * CNAME Records

* Infrastructure Discovery

  * Domain Metadata
  * Server Information
  * DNS Mapping

---

### 🕵️ Person Investigation

#### Web Intelligence

## ⚡ Installation

### Clone Repository

git clone https://github.com/yourusername/CyberSentinel.git

### Frontend Setup

cd frontend

npm install

npm run dev

### Backend Setup

cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
